## üöÄ Features

### Core Capabilities
- **Automated Data Analysis**: Complete EDA, statistical testing, and insights generation
- **Multi-file Support**: Analyze relationships between multiple datasets
- **AI Assistant**: ChatGPT-powered assistant for queries, SQL generation, and code help
- **Automated Reporting**: Generate comprehensive analysis reports in multiple formats

### Data Processing
- **Data Cleaning**: Handle missing values, remove duplicates, correct data types
- **Outlier Detection**: Identify and handle outliers using statistical methods
- **Feature Engineering**: Create derived metrics and aggregate features
- **Data Validation**: Comprehensive data quality assessment

### Analysis Modules
- **Exploratory Data Analysis (EDA)**: Distribution analysis, correlation studies, trend identification
- **Statistical Testing**: Normality tests, hypothesis testing, ANOVA, non-parametric tests
- **Business Insights**: KPI dashboards, performance metrics, strategic recommendations
- **Multi-file Analysis**: Schema analysis, relationship mapping, integration recommendations

### AI Integration
- **Smart Assistant**: Answer questions about your data and analysis
- **Code Generation**: Generate SQL queries and Python code for analysis
- **Analysis Ideas**: Get suggestions for further exploration
- **Report Enhancement**: AI-powered insights and recommendations
